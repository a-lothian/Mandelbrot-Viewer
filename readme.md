# MandelBrot Set Viewer in C

_A simple Mandelbrot set viewer written in C using SDL2._

This project was primarily a **Github Actions** learning excersise, including:

- Multi-platform builds (macOS, Linux, Windows)
- Automated releases
- CI troubleshooting across toolchains

The program itself isn't particulary special.

![Image showcase](demo.png)

## Features

- Cross-platform (macOS, Linux, Windows)
- Mouse controls: Dragging, Zooming
- Built and released automatically via GitHub Actions
- Not written by AI
